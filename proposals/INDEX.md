# Proposal index

The catalog of every proposal merged into `main` — released, rejected, and superseded alike. The **proposal number** is assigned here, at merge; it appears nowhere else, and no proposal directory is ever renamed. Numbers run in a single sequence across all types and outcomes.

A proposal earns its number only once it is decided and merged. While a proposal is still open — `#proposed` or `#accepted` — it lives at `proposals/<slug>/` with no number, and does not appear here.
