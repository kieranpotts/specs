# Compatibility

## Web browser support

The client application MUST be distributable as a bundle compatible with the following mainstream web browsers, at the given versions or later (release dates in parentheses):

- Chrome vXX (MM YYYY)
- Firefox vXX (MM YYYY)
- Safari vXX (MM YYYY)

_Set concrete minimum versions and review them on a defined cadence. A common policy is to support the latest two major versions of each browser._
