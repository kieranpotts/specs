# Skills

The following skills are defined in this repository for managing the proposal workflow via agentic tools:

- [`draft-proposal`](./draft-proposal/): Scaffold a new proposal branch and document.

- [`check-proposal`](./check-proposal/): Audit a proposal for completeness and process compliance before advancing it.

- [`advance-proposal`](./advance-proposal/): Transition a proposal to the next permitted lifecycle state.

- [`reject-proposal`](./reject-proposal/): Handle the rejection path — revert spec edits and prepare the PR for merge.

- [`index-proposals`](./index-proposals/): Regenerate the index table in `proposals/README.md`.
